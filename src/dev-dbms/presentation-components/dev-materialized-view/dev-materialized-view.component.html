Materialized View template.


This is a template. It's not rendered right away. It is rendered after the view is initialized.


This template could be referenced by its local variable
#materializedViewTemplate using @ViewChild to query for this
local variable in a Service and pass this template by reference to
a different component after the view is initialized.
Rendering a template involves the creation of a view.
A view is rendered HTML.
View containers create views.
A view can be created dynamically in Angular using view containers.
A view container is a location in the DOM that lets you instantiate templates
or components inside of it.
View containers will keep track of the templates or components for you and
make sure change detection works,
as well as remove the component or template when it is time to destroy them.
